AsistenciaApp

With this application, you can manage employee attendance for small companies. You can register in a local database using a QR code the date, local time, and the associated name. The application offers the following features:

  • CRUD operations
  • Sending an email notification when a QR code is registered
  • User management
  • QR code generation based on the ID parameter
  • Creation of an Excel file containing all registrations
  • Support for all cameras (Windows)
